
=================================================
GUIDE D'UTILISATION RAPIDE DE FILECHAT
=================================================

Ce dossier contient FileChat, votre assistant d'intelligence documentaire.
Voici comment l'utiliser facilement:

1. DÉMARRAGE RAPIDE
-------------------
Double-cliquez sur "demarrer-filechat.bat" pour lancer l'application.
Une fenêtre de commande s'ouvrira, puis FileChat s'ouvrira dans votre navigateur.

2. UTILISATION AVEC IA LOCALE
----------------------------
Pour utiliser l'IA locale (plus rapide, sans connexion internet nécessaire):
1. Installez Ollama depuis https://ollama.ai/download
2. Double-cliquez sur "demarrer-ia-locale.bat"
3. Suivez les instructions à l'écran pour configurer l'IA locale

3. EN CAS DE PROBLÈME
--------------------
Si vous rencontrez des problèmes (page blanche, erreurs, etc.):
1. Double-cliquez sur "reparer-filechat.bat"
2. Attendez que la réparation soit terminée
3. Relancez FileChat avec "demarrer-filechat.bat"

REMARQUES IMPORTANTES:
- Ne fermez pas les fenêtres de commande pendant l'utilisation
- Pour quitter, fermez la fenêtre de commande principale
- En cas de problème persistant, contactez le support technique
