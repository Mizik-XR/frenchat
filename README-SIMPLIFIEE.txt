
=================================================
GUIDE D'UTILISATION SIMPLIFIÉE DE FILECHAT
=================================================

Bienvenue dans la version simplifiée de FileChat.
Ce guide vous aidera à utiliser l'application sans problèmes techniques.

1. INSTALLATION PROPRE
-------------------
Si vous rencontrez des problèmes, lancez d'abord une installation propre:
Double-cliquez sur "nettoyer-et-installer.bat"
Ce script supprimera tous les fichiers temporaires et réinstallera l'application.

2. DÉMARRAGE SIMPLE
-------------------
Double-cliquez sur "demarrer-filechat-simple.bat"
Une fenêtre de commande s'ouvrira, puis FileChat s'ouvrira dans votre navigateur.
Ne fermez pas la fenêtre de commande pendant l'utilisation.

3. PRÉPARATION POUR NETLIFY
--------------------------
Une fois que tout fonctionne comme prévu:
Double-cliquez sur "preparer-pour-netlify.bat"
Ce script préparera l'application pour un déploiement sur Netlify.

REMARQUES IMPORTANTES:
- Cette version n'utilise que le mode Cloud (pas d'IA locale)
- Tous les problèmes d'installation et de démarrage devraient être résolus
- Pour quitter, fermez la fenêtre de commande principale
